## Main spine — ordered concept map

### Beginner foundation course (17 modules, 178 lessons)

| Module group | Ordered concept coverage (labels only) | Pacing / granularity note |
|---|---|---|
| Opening modules | course orientation → **hiragana** (introduced before any grammar), basic pronunciation | Slow, fine-grained: a full module just for kana onboarding |
| Early grammar | greetings/set expressions, **だ/です copula**, basic **question (か)**, **particle は / が**, first short reading practice | Phrase-first: usable sentences appear before deep grammar |
| Script expansion | **katakana** introduced as its own module (after learners already have spoken/grammar footing) | Katakana deliberately delayed relative to hiragana |
| Core sentence building | **demonstratives/pronouns**, **negatives**, **numbers**, more particles | Grouped: pronouns + negation + numbers taught close together |
| Verbs enter | **verb groups**, **-ます/polite form**, **particle を**, adverbs | Polite register taught first as the default verb form |
| Kanji begins (~module 6) | **kanji** introduced and then threaded through nearly every subsequent module alongside grammar | Kanji is interleaved, not front-loaded or fully deferred |
| Mid grammar | **て-form**, **て-いる/progressive**, expanded particles, more verb behavior | te-form is a hinge point, taught once polite verbs are solid |
| Adjective + comparison | **i/na adjectives**, negation of adjectives, **comparatives** | Adjectives arrive relatively late vs. verbs |
| Advanced beginner | **potential form**, **imperative**, **passive**, **causative** | Reaches the harder voice/mood transforms by the end |
| Closing | **dates / time / calendar** as a capstone practical module | Practical time-vocabulary parked at the end rather than early |

**Granularity:** very fine — 178 lessons for what is essentially the N5→low-N4 band, with kana and pronunciation each given their own multi-lesson runway.

### Intermediate course (13 modules, 94 lessons)

- Built around **graded reading + listening**: most lessons are text-driven rather than titled by a single grammar point.
- Grammar (additional verb behavior, polite-form review, adjective review, negation, pronunciation refinement) is **woven into texts** instead of taught as isolated grammar lessons.
- **Granularity:** medium-coarse — grammar is contextual/implicit, so the explicit topic labels thin out compared with the beginner course.

### Advanced course (12 modules, 85 lessons)

- Same reading-/listening-first model as intermediate, at higher difficulty.
- Surfaced concept labels are sparse (verbs, particle を, polite review, dates/time) because content is delivered as longer texts with grammar embedded.
- **Granularity:** coarse — assumes the learner consolidates grammar through volume of input rather than discrete lessons.

## Dedicated kanji track (16 modules, 50 lessons)

- Highly **regular cadence**: almost every module is exactly 3 lessons (an intro/teaching unit, a short pivot unit, and a larger practice/worksheet unit), with **downloadable practice sheets** attached.
- Runs **in parallel** to the grammar spine rather than inside it — a learner does kanji as a separate, drillable stream.
- **Granularity:** batch-based — kanji delivered in fixed-size sets module by module.
